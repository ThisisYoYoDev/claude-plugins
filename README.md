# YoyoDev Marketplace

Claude Code plugin marketplace by [Yoel Edery](https://www.linkedin.com/in/yoel-edery/).

## Plugins

| Plugin | Description |
|--------|-------------|
| **[claude-rag](https://github.com/ThisisYoYoDev/claude-rag)** | RAG plugin — capture, index, and search across all your conversations. [clauderag.io](https://clauderag.io) |

## Install

Add this marketplace to Claude Code:

```
/plugin marketplace add ThisisYoYoDev/yoyodev-marketplace
```

Then install a plugin:

```
/plugin install claude-rag@yoyodev
```
